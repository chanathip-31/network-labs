# Network Engineering Lab 

Personal network engineering lab collection built with Cisco Packet Tracer and Proxmox.  
Covers enterprise switching, routing, security, wireless, and virtualization infrastructure.

---

## Lab Topics

### 1. Foundation Switching
| Lab | Topic |
|-----|-------|
| LAB_VLAN_and_TRUNK | VLAN segmentation and trunk link configuration |
| LAB_STP | Spanning Tree Protocol |
| LAB_Rapid_STP | Rapid Spanning Tree Protocol |
| LAB_PortFast_BPDU_Guard | PortFast and BPDU Guard |
| LAB_EtherChannel_PAgP | EtherChannel with PAgP |
| LAB_EtherChannel_LACP | EtherChannel with LACP |

### 2. Routing & Connectivity
| Lab | Topic |
|-----|-------|
| LAB_Static_Route | Static routing configuration |
| LAB_Default_Route | Default route configuration |
| LAB_Single_Area_OSPF | OSPF Single Area |
| LAB_Multi_Area_OSPF | OSPF Multi-Area |
| LAB_ROAS | Inter-VLAN Routing via Router-on-a-Stick |
| LAB_SVI | Inter-VLAN Routing via Switched Virtual Interface |

### 3. Security & Services
| Lab | Topic |
|-----|-------|
| LAB_Standard_ACL | Standard Access Control List |
| LAB_Extended_ACL | Extended Access Control List |
| LAB_Static_NAT | Static Network Address Translation |
| LAB_PAT_Overload | PAT Overload |
| LAB_Router_DHCP | DHCP Server on Router |
| LAB_Core_L3_Switch_DHCP | DHCP Server on Layer 3 Switch |
| LAB_DHCP_Relay | DHCP Relay Agent |

### 4. Wireless Infrastructure
| Lab | Topic |
|-----|-------|
| LAB_Wireless_AP | Wireless Access Point configuration |
| LAB_WLC | Wireless LAN Controller configuration |

### 5. Proxmox Lab
| Component | Details |
|-----------|---------|
| Proxmox VE Cluster | 2-node cluster with HA failover |
| FortiGate VM | Virtual IPs, Policy Routing, NAT |
| Ubuntu Server | Docker containers, self-hosted services |
| Windows Server 2022 | AD DS, OU structure, user management |
| Tailscale VPN | Secure remote connectivity |
